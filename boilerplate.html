What is version control

Revision control, also known as version control and source control (and an aspect of software configuration management), is the management of changes to documents, computer programs, large web sites, and other collections of information. - Wikipedia

Why Should I care ?

  • revert files back to a previous state ( Cntrl+Z may not always do )

  • revert the entire project back to a previous state

  • compare changes over time

  • see who last modified something

  • Coordinating Teams

Summary : If you screw things up or lose files, you can easily recover.

Design Goals

  • Speed
  • Simple design
  • Strong support for non-linear development (thousands of parallel branches)
  • Fully distributed
  • Able to handle large projects like the Linux kernel efficiently (speed and data size)
  • Compatibility with existing systems/protocols
    • HTTP
    • FTP
    • Fileshare
    • SSH
    • and more ..
  • Periodic explicit object packing
  • Automatic Garbage collection
